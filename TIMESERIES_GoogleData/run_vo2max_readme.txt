Time Series Data Export

The Time Series export provides a detailed timeline of your tracked activity.

Files Included:
----------

run_vo2max_YYYY-MM-DD.csv             - Where YYYY-MM-DD is the starting date for the entries in the file

Run VO2 max measures a user's maximal oxygen consumption while running
Each entry has the following values

    timestamp                         - Date and time at which the entry was logged
    run VO2 max                       - VO2 Max value in ml/kg/min
